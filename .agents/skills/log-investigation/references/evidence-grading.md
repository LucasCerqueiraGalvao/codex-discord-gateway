# Evidence Grading

Usar esta rubrica para evitar conclusoes exageradas.

## Alta confianca

Usar quando houver:

- erro ou timeout diretamente ligado ao fluxo afetado
- correlacao temporal clara
- identificador compartilhado entre eventos
- comportamento observado compatível com o sintoma

Resposta esperada:

- nomear a hipotese principal
- mostrar 2 ou 3 evidencias objetivas
- apontar o proximo passo mais direto

## Media confianca

Usar quando houver:

- sintomas consistentes, mas sem identificador forte
- correlacao temporal boa, porem com lacunas
- varias pistas apontando para a mesma area

Resposta esperada:

- apresentar a melhor hipotese com cautela
- dizer o que falta confirmar
- sugerir a melhor leitura complementar

## Baixa confianca

Usar quando houver:

- logs incompletos ou ruidosos
- multiplas causas plausiveis
- falta de janela temporal ou de correlacao

Resposta esperada:

- dizer claramente que a evidência ainda e insuficiente
- evitar chamar de causa raiz
- pedir ou sugerir a próxima fonte certa de dado
